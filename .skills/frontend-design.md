# Frontend Design Skill

> Source: anthropics/skills — `skills/frontend-design/SKILL.md`

## Core Philosophy

This skill enables creation of **distinctive, production-grade frontend interfaces** that reject generic AI aesthetics. Build functional, visually striking web components, pages, and applications.

Emphasize **intentional design direction** before implementation:

- Understand purpose, audience, and technical constraints
- Commit to a bold aesthetic (minimalist, maximalist, retro, brutalist, etc.)
- Execute with precision and cohesion

> "Choose a clear conceptual direction and execute it with **precision**. Bold maximalism and refined minimalism both work — the key is intentionality, not intensity."

## Design Priorities

### Typography
Distinctive, characterful fonts. Pair display and body selections thoughtfully rather than defaulting to common choices.

### Color & Motion
Cohesive palettes with CSS variables. Strategic animations and scroll-triggered effects that create high-impact moments.

### Spatial Composition
Embrace asymmetry, overlap, unexpected layouts, and generous negative space.

## Anti-Patterns

Explicitly reject:
- Generic fonts (Arial, Inter, Roboto) without justification
- Cliched color schemes (particularly purple gradients)
- Predictable layouts
- Cookie-cutter design that lacks context-specific character

**Critical mandate**: Vary aesthetic approaches across projects. Never converge on overused solutions.
